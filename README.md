# Mental Health Predictive Analytics Project

## Overview
This project explores how lifestyle and behavioral factors relate to stress levels using a mental health dataset from Kaggle. The goal was to apply data analysis and machine learning techniques to better understand what factors may influence stress.

## Objectives
- Explore relationships between lifestyle variables and stress
- Build predictive models to classify stress level
- Compare model performance
- Identify key features influencing stress

## Dataset
Kaggle Mental Health Dataset:  
https://www.kaggle.com/datasets/jajidhasan/mental-health

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Models Used
- Logistic Regression
- Random Forest

## Results
Both models showed relatively low accuracy, indicating that stress is difficult to predict using the available variables. However, feature importance analysis highlighted key factors such as screen time, sleep, and work/study hours.

## How to Run
1. Open the notebook in Google Colab
2. Upload the dataset file
3. Run all cells in order

## Key Takeaways
- Stress is influenced by multiple lifestyle factors
- Predicting stress is complex and not highly accurate with basic data
- Lifestyle habits like sleep and screen time still show meaningful patterns
